# 🛍️ NxtTrendz – E-Commerce Web Application

NxtTrendz is a modern and responsive e-commerce web application inspired by leading online shopping platforms like Amazon and Flipkart. The application provides a seamless shopping experience with secure authentication, product browsing, advanced filtering, and cart management.

## 🚀 Live Demo

https://nxt-trendz-kohl-five.vercel.app/

---

## ✨ Features

* 🔐 Secure User Authentication (JWT)
* 🛒 Add, Update, and Remove Cart Items
* 📦 Browse Products with Detailed Information
* 🔍 Search Products by Name
* 🎯 Filter Products by Category and Rating
* ↕️ Sort Products by Price
* 📱 Fully Responsive Design
* ⚡ Fast API Integration
* 🚫 Protected Routes for Authenticated Users
* 🍪 Persistent Login using JWT Tokens

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* JavaScript (ES6+)
* HTML5
* CSS3
* Axios

### Backend

* Node.js
* Express.js
* JWT Authentication
* REST APIs

### Database

* Supabase (PostgreSQL)

### Deployment

* Frontend: Vercel
* Backend: Render

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/NxtTrendz.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
